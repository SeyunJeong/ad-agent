# KPI 정의서

## 개요
퍼포먼스 마케팅 팀에서 사용하는 모든 KPI의 정의, 계산 방법, 채널별 벤치마크를 정리한다.
모든 에이전트는 이 정의를 기준으로 성과를 측정하고 보고한다.

---

## 핵심 KPI 목록

### Tier 1: 비즈니스 KPI (최종 목표)

| KPI | 정의 | 계산식 | 단위 |
|-----|------|--------|------|
| **ROAS** (광고수익률) | 광고비 대비 매출 | 매출 / 광고비 x 100 | % 또는 배수 |
| **ROI** (투자수익률) | 순이익 기준 수익률 | (매출 - 광고비 - 원가) / 광고비 x 100 | % |
| **매출** | 광고 기여 매출 | 전환 가치 합산 | 원 |
| **전환수** | 목표 행동 완료 횟수 | 구매/가입/문의 등 카운트 | 건 |

### Tier 2: 효율 KPI (비용 효율)

| KPI | 정의 | 계산식 | 단위 |
|-----|------|--------|------|
| **CPA** (전환당 비용) | 전환 1건 확보 비용 | 총 비용 / 전환수 | 원/건 |
| **CPC** (클릭당 비용) | 클릭 1회 비용 | 총 비용 / 클릭수 | 원/회 |
| **CPM** (1000회 노출 비용) | 1000회 노출 비용 | 총 비용 / 노출수 x 1000 | 원 |
| **CPL** (리드당 비용) | 리드 1건 확보 비용 | 총 비용 / 리드수 | 원/건 |
| **CPI** (설치당 비용) | 앱 설치 1건 비용 | 총 비용 / 설치수 | 원/건 |

### Tier 3: 퍼널 KPI (단계별 효율)

| KPI | 정의 | 계산식 | 단위 |
|-----|------|--------|------|
| **CTR** (클릭률) | 노출 대비 클릭 비율 | 클릭수 / 노출수 x 100 | % |
| **CVR** (전환율) | 클릭 대비 전환 비율 | 전환수 / 클릭수 x 100 | % |
| **도달률** | 타겟 모수 대비 도달 | 도달수 / 타겟 모수 x 100 | % |
| **빈도** | 1인당 평균 노출 횟수 | 노출수 / 도달수 | 회 |
| **이탈률** | 랜딩 후 즉시 이탈 | 이탈수 / 방문수 x 100 | % |

### Tier 4: 기타 KPI

| KPI | 정의 | 계산식 | 단위 |
|-----|------|--------|------|
| **노출수** | 광고가 화면에 표시된 횟수 | 채널 집계 | 회 |
| **도달수** | 광고를 본 고유 사용자 수 | 채널 집계 | 명 |
| **클릭수** | 광고를 클릭한 횟수 | 채널 집계 | 회 |
| **체류 시간** | 랜딩 페이지 평균 체류 | GA4 집계 | 초 |
| **페이지뷰/세션** | 세션당 페이지 조회수 | GA4 집계 | 회 |
| **예산 소진율** | 예산 대비 실제 지출 | 실지출 / 예산 x 100 | % |

## KPI 계산 상세

### ROAS 계산

```
기본 ROAS = 광고 기여 매출 / 광고비

예시:
├── 광고비: 1,000,000원
├── 광고 기여 매출: 5,000,000원
├── ROAS = 5,000,000 / 1,000,000 = 500% (5배)
│
├── ROAS 500% = 1원 투자 시 5원 매출
├── ROAS 300% = 1원 투자 시 3원 매출
└── ROAS 100% = 1원 투자 시 1원 매출 (본전)

손익분기 ROAS 계산:
├── 매출총이익률 50% → 최소 ROAS 200%
├── 매출총이익률 40% → 최소 ROAS 250%
├── 매출총이익률 30% → 최소 ROAS 333%
└── 매출총이익률 20% → 최소 ROAS 500%

공식: 손익분기 ROAS = 1 / 매출총이익률 x 100
```

### CPA 계산

```
CPA = 총 광고비 / 전환수

예시:
├── 광고비: 500,000원
├── 전환수: 50건
├── CPA = 500,000 / 50 = 10,000원/건
│
목표 CPA 설정:
├── 평균 객단가: 50,000원
├── 매출총이익률: 40%
├── 매출총이익: 20,000원
├── 목표 이익률 10%: 목표 CPA = 20,000 - 5,000 = 15,000원
└── 손익분기 CPA = 매출총이익 = 20,000원
```

### CTR/CVR 해석 가이드

```
CTR (클릭률):
├── 높은 CTR: 광고 관련성/매력도 높음
├── 낮은 CTR: 키워드 미스매치, 소재 비매력
└── 비정상 높은 CTR: 브랜드 키워드 or 클릭 어뷰즈 의심

CVR (전환율):
├── 높은 CVR: 랜딩페이지 최적화, 타겟 정확
├── 낮은 CVR: LP 문제, 타겟 미스매치, 가격 경쟁력
└── CTR 높은데 CVR 낮음: LP 최적화 필요 (기대 vs 현실 갭)
```

## KPI 대시보드 기본 구성

### 경영진 뷰 (요약)
```
핵심 지표 4개: ROAS, CPA, 총 전환수, 총 광고비
기간 비교: 전주/전월 대비 변화
상태 표시: 정상(녹색)/주의(노랑)/위험(빨강)
```

### 매니저 뷰 (채널별)
```
채널별 핵심 지표: CTR, CPC, CPA, ROAS
캠페인별 성과 순위
예산 소진율
```

### 운영자 뷰 (상세)
```
키워드/소재별 성과
시간대별/요일별 성과
오디언스별 성과
입찰 변동 이력
```

## KPI 목표 설정 가이드라인

### SMART 기준
```
S (Specific): "ROAS 400% 달성" (O) / "성과 좋게" (X)
M (Measurable): 정량적으로 측정 가능한 수치
A (Achievable): 벤치마크 기반 현실적 목표
R (Relevant): 비즈니스 목표와 직접 연결
T (Time-bound): "3월 말까지" 등 기한 명시
```

### 목표 설정 프로세스
```
1. 비즈니스 목표 확인 (매출 X원 달성)
2. 필요 전환수 역산 (매출 / 객단가)
3. 필요 클릭수 역산 (전환수 / 예상 CVR)
4. 필요 예산 산출 (클릭수 x 예상 CPC)
5. 목표 CPA/ROAS 계산
6. 채널별 벤치마크와 비교하여 현실성 검증
7. 최종 KPI 목표 확정
```
