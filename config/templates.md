# 캠페인 템플릿

## 개요
업종별로 검증된 캠페인 기본 설정을 템플릿으로 제공한다.
신규 캠페인 생성 시 해당 업종의 템플릿을 기반으로 시작하면 학습 기간을 단축하고 초기 성과를 높일 수 있다.

> **주의**: 템플릿은 출발점이지 정답이 아니다. 반드시 해당 비즈니스의 특성에 맞게 커스터마이즈해야 한다.

---

## 1. 이커머스 (일반 쇼핑몰)

### 캠페인 개요
```
목표: 매출 극대화 (ROAS 중심)
주력 채널: 네이버 쇼핑검색 > 메타 > 구글 쇼핑 > 카카오
예산 배분: 네이버 40% / 메타 30% / 구글 20% / 카카오 10%
핵심 KPI: ROAS 300~500%, CPA {객단가의 30% 이하}
```

### 채널별 전략
```
네이버 쇼핑검색:
├── 상품 카탈로그 최적화 (상품명, 이미지, 가격)
├── 카테고리별 입찰 그룹 분리
├── 고마진 상품 우선 입찰
└── 리뷰수/평점 높은 상품 집중

네이버 파워링크:
├── 브랜드 키워드 방어
├── 카테고리 키워드 (예: "운동화 추천")
├── 롱테일 키워드 (예: "나이키 에어맥스 270 블랙")
└── 구매 의도 키워드 집중

메타 (인스타그램):
├── DPA (다이나믹 상품 광고): 리타겟팅
├── 신규 고객: 유사 오디언스 (구매자 기반 1~3%)
├── 세일/프로모션: 캐러셀 광고
└── 리일스: 제품 활용 영상

구글:
├── 쇼핑 캠페인 또는 P-Max
├── 검색: 구매 의도 키워드
├── 디스플레이: 리마케팅
└── YouTube: 제품 소개 영상

카카오:
├── 친구톡: 구매 고객 리텐션
├── 디스플레이: 인지도 보조
└── 비즈보드: 대규모 프로모션 시
```

### 퍼널 전략
```
TOF (인지): 메타 리치 광고, 카카오 비즈보드
MOF (고려): 네이버 검색, 메타 트래픽, 구글 검색
BOF (전환): 네이버 쇼핑, 메타 DPA, 구글 쇼핑
리텐션: 카카오 친구톡, 메타 리타겟팅
```

## 2. 이커머스 (패션/뷰티)

### 캠페인 개요
```
목표: 매출 + 브랜드 인지
주력 채널: 메타 (인스타그램) > 네이버 > 카카오 > 구글
예산 배분: 메타 40% / 네이버 30% / 카카오 15% / 구글 15%
핵심 KPI: ROAS 200~400%, CTR 2% 이상
```

### 채널별 전략
```
메타 (인스타그램 메인):
├── 피드: 룩북/스타일링 이미지
├── 리일스: 착용 영상, 언박싱
├── 스토리: 한정 세일, 신상 공개
├── DPA: 상품 조회/장바구니 리타겟팅
├── 인플루언서 콘텐츠 부스팅
└── 유사 오디언스: 구매자 기반

네이버:
├── 쇼핑검색: 상품명 SEO 최적화
├── 파워링크: "{브랜드} + 아이템" 키워드
├── 브랜드검색: 브랜드 인지도 높을 때
└── 파워콘텐츠: 스타일 가이드/후기

카카오:
├── 비즈보드: 시즌 캠페인
├── 디스플레이: 신상 알림
└── 친구톡: VIP/리피터 타겟

구글:
├── YouTube: 브랜드 영상, 스타일링 콘텐츠
├── 디스플레이: 리마케팅
└── 쇼핑: 보조 채널
```

### 특수 사항
```
- 비주얼 퀄리티가 CTR에 직접 영향 → thread-team 연동 필수
- 시즌 변화에 민감 → 시즌별 소재 교체 스케줄 수립
- 인플루언서 마케팅과 페이드 광고 시너지 설계
- 할인 시즌 (블프, 연말) 예산 2~3배 증액 고려
```

## 3. SaaS / B2B

### 캠페인 개요
```
목표: 리드 생성 (회원가입, 무료체험, 데모 신청)
주력 채널: 구글 검색 > 네이버 검색 > 메타 > 카카오
예산 배분: 구글 35% / 네이버 35% / 메타 20% / 카카오 10%
핵심 KPI: CPL, 리드 → 전환율, LTV:CAC 비율
```

### 채널별 전략
```
구글 검색:
├── 문제 인식 키워드: "~하는법", "~솔루션"
├── 제품 카테고리 키워드: "{카테고리} 소프트웨어"
├── 경쟁사 키워드: "{경쟁사} 대안"
├── 브랜드 키워드: 방어
└── 확장: Performance Max (B2B 리드용)

네이버 검색:
├── 정보 탐색 키워드: "{분야} 프로그램 추천"
├── 비교 키워드: "{A} vs {B}"
├── 파워콘텐츠: 솔루션 소개, 사례 연구
└── 파워링크: 직접 전환 유도

메타:
├── 리드 생성 광고 (인앱 폼)
├── 콘텐츠 마케팅 부스팅 (백서, 웨비나)
├── 직책/산업 기반 타겟팅 (B2B)
├── 유사 오디언스: 기존 고객 기반
└── 리타겟팅: 웹사이트 방문자

카카오:
├── 디스플레이: 비즈니스/IT 관심사 타겟
├── 알림톡: 리드 후 너처링
└── 보조 채널로 활용
```

### 특수 사항
```
- 전환 사이클이 길음 (평균 30~90일) → 리드 스코어링 필요
- LTV(고객 생애 가치) 기반 CPL 목표 설정
- 콘텐츠 마케팅과 유기적 연계 (백서, 블로그, 웨비나)
- 데모 신청 → 세일즈 핸드오프 추적 필수
- 구글이 한국 B2B 검색에서 네이버보다 비중 높을 수 있음
```

## 4. 로컬 비즈니스 (음식점, 매장, 서비스)

### 캠페인 개요
```
목표: 방문/문의/예약 유도
주력 채널: 네이버 > 카카오 > 메타 > 구글
예산 배분: 네이버 40% / 카카오 25% / 메타 20% / 구글 15%
핵심 KPI: CPA (방문/문의당 비용), 지역 도달수
```

### 채널별 전략
```
네이버:
├── 네이버 플레이스 최적화 (필수, 무료)
├── 파워링크: 지역 + 업종 키워드 ("강남 맛집", "홍대 미용실")
├── 쇼핑검색: 온라인 판매 있을 경우
├── 브랜드검색: 인지도 높은 매장
└── 스마트플레이스 광고

카카오:
├── 카카오맵 광고 (지역 검색)
├── 디스플레이: 반경 타겟팅
├── 친구톡: 단골 고객 재방문 유도
└── 알림톡: 예약 확인/리마인더

메타:
├── 도달/인지도 캠페인: 반경 타겟팅
├── 인스타그램: 매장/메뉴 사진/영상
├── 스토리: 일일 메뉴, 이벤트
└── 리타겟팅: 페이지 참여자

구글:
├── 구글 비즈니스 프로필 최적화 (필수, 무료)
├── 검색: 지역 + 업종 키워드
├── 디스플레이: 반경 타겟팅
└── YouTube: 매장 소개 영상 (선택)
```

### 특수 사항
```
- 반경 타겟팅 필수 (매장 주변 1~5km)
- 리뷰/평점 관리가 광고 효율에 직접 영향
- 요일/시간대별 입찰 조정 (주말 높게, 영업시간 집중)
- 네이버 플레이스/구글 비즈니스 프로필 최적화가 광고보다 우선
- 소규모 예산(월 50~200만원)에 최적화된 전략 필요
```

## 5. 앱 서비스 (모바일 앱)

### 캠페인 개요
```
목표: 앱 설치 + 활성화 (인앱 행동)
주력 채널: 메타 > 구글 UAC > 카카오 > 네이버
예산 배분: 메타 35% / 구글 30% / 카카오 20% / 네이버 15%
핵심 KPI: CPI (설치당 비용), CPA (활성화당 비용), D7 리텐션
```

### 채널별 전략
```
메타:
├── 앱 설치 캠페인 (자동 최적화)
├── 앱 이벤트 최적화 (가입/구매 등)
├── 리일스/스토리: 앱 활용 영상
├── 유사 오디언스: 활성 사용자 기반
└── 앱 리타겟팅: 비활성 사용자 재유도

구글 (UAC/ACe):
├── Universal App Campaign (전체 구글 인벤토리)
├── App Campaigns for Engagement (리타겟팅)
├── 에셋: 텍스트 + 이미지 + 영상
└── ML 기반 자동 최적화

카카오:
├── 앱 설치 캠페인
├── 카카오톡 채널: 유입 후 앱 설치 유도
├── 디스플레이: 앱 프로모션
└── SDK 연동: 인앱 이벤트 추적

네이버:
├── 파워링크: "{앱명}", "{카테고리} 앱"
├── 파워콘텐츠: 앱 소개/리뷰
└── 보조 채널 (검색 유입)
```

### 특수 사항
```
- MMP (Mobile Measurement Partner) 필수: AppsFlyer, Adjust, Branch 등
- iOS ATT 대응 → SKAN(SKAdNetwork) 설정 필수
- 인앱 이벤트 추적 → 설치 이후 행동까지 측정
- 리텐션 KPI 중요: D1, D7, D30 리텐션
- 프리미엄 vs 프리미엄 유저 구분 (구매자 최적화)
```

## 6. 교육/강의 (온라인 교육)

### 캠페인 개요
```
목표: 수강 신청 / 무료 강의 체험
주력 채널: 네이버 > 메타 > 카카오 > 구글
예산 배분: 네이버 40% / 메타 25% / 카카오 20% / 구글 15%
핵심 KPI: CPA (수강 신청), CTR, LP 전환율
```

### 채널별 전략
```
네이버:
├── 파워링크: "{분야} 강의", "{자격증} 학원"
├── 파워콘텐츠: 무료 콘텐츠 → 유료 전환
├── 브랜드검색: 유명 강사/브랜드
└── 쇼핑검색: 교재/도서 판매 시

메타:
├── 리드 생성: 무료 강의/자료 다운로드
├── 영상 광고: 강의 맛보기
├── 인스타그램: 강사 개인 브랜딩
├── 리타겟팅: 방문자/영상 조회자
└── 유사 오디언스: 기존 수강생 기반

카카오:
├── 디스플레이: 관심사(교육/자격증) 타겟
├── 친구톡: 기존 수강생 업셀
├── 알림톡: 수강 리마인더, 개강 알림
└── 비즈보드: 대규모 모집 시

구글:
├── 검색: 정보 탐색 키워드 "{분야} 배우기"
├── YouTube: 강의 프리뷰 영상
├── 디스플레이: 리마케팅
└── 보조 채널
```

### 특수 사항
```
- 무료 체험/맛보기 → 유료 전환 퍼널 설계 필수
- 시즌성: 1월(신년 목표), 3월(새 학기), 9월(하반기) 피크
- 수강 후기/합격 사례가 전환에 큰 영향
- 교육 광고 규제: 학원 등록번호, 과대광고 주의
```

## 템플릿 사용 가이드

### 커스터마이즈 체크리스트
```
[ ] 비즈니스 고유 USP 반영
[ ] 실제 예산에 맞게 채널/비율 조정
[ ] 타겟 오디언스 구체화
[ ] 업종 벤치마크 최신 데이터 서치
[ ] 경쟁사 분석 결과 반영
[ ] 시즌/이벤트 일정 반영
[ ] 기존 자산(소재/고객 데이터) 활용 방안
[ ] KPI 목표 SMART 기준 설정
```
