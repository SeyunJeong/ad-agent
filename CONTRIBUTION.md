## 팀별 기여 상세

### 1. biz-planner (기획부) — 사업 설계

| 항목 | 내용 |
|------|------|
| **기여 역량** | 사업 전략 수립, 시장 분석, 제품 설계 |
| **산출물** | `BUSINESS_PLAN.md` (911줄) |
| **핵심 성과** | |
| - 시장 분석 | 한국 온라인 광고 시장 10.1조 원 규모 실데이터 기반 분석, 출처 16개 |

| - 수익 모델 | 구독+수수료 하이브리드 모델, 12개월 매출 예측 (연 6.5억) |
| - API 스펙 | 플러그인 아키텍처 초안, REST API 엔드포인트 정의, JSON 예시 |
| - 로드맵 | MVP→V1→V2 3단계 로드맵, 리스크 분석 포함 |
| **평가** | A — 실데이터 기반 + TAM/SAM/SOM 산출 + 한국 특화 분석 우수 |

### 3. thread-team (마케팅부) — 크리에이티브 역량 제공

| 항목 | 내용 |
|------|------|
| **기여 역량** | 광고 카피 생성 노하우, 콘텐츠 파이프라인 연동 |
| **기여 방식** | 직접 빌드가 아닌 **역량 연동** — AD_CREATIVE 에이전트 + handoff/creative_to_thread.md를 통해 통합 |
| **연동 포인트** | |
| - 크리에이티브 소재 | 고품질 광고 소재가 필요할 때 thread-team에 제작 요청 |
| - 카피 가이드 | 한국어 카피라이팅 노하우 AD_CREATIVE 에이전트에 반영 |
| - 콘텐츠 재활용 | 오가닉 콘텐츠를 광고 소재로 전환하는 파이프라인 |
| **평가** | B+ — 직접 빌드 참여는 아니나, 핵심 연동 포인트로 활용도 높음 |
