```
   └── GA4 전환 이벤트를 Google Ads로 가져오기

3. GA4 내 설정
   ├── 전환 이벤트 마킹: first_open, first_link_save
   ├── 맞춤 차원: utm_source, utm_medium, utm_campaign, utm_content
   ├── 오디언스 빌더: 설치 후 3일 내 미활성 유저 (리인게이지먼트용)
   └── 탐색(Explorations): 전환 퍼널 설정
       └── 앱 설치 → 첫 실행 → 첫 링크 저장 → AI 요약 조회

4. 데이터 교차 검증
   ├── Google Ads 설치 수 vs GA4 first_open 비교
   ├── 허용 오차: 10% 이내
   └── 초과 시: 전환 추적 코드 점검
```

### 4.5 YouTube 광고 배치 포함 여부

> **포함 (UAC 자동 배분)**
> Google App Campaign(UAC)은 YouTube를 포함한 전체 구글 네트워크에 자동 배치. 별도 YouTube 캠페인을 만들지 않고, UAC 내에서 영상 에셋(가로/세로/정사각)을 제공하면 YouTube 인스트림, 인피드, 쇼츠에 자동 게재됨.
> - YouTube 쇼츠: 9:16 세로 영상 에셋이 자동 게재
> - YouTube 인스트림: 16:9 가로 영상 에셋이 자동 게재
> - 한국 YouTube MAU 4,500만+ 활용하여 IT/테크 관심 유저 도달
