| 단계 | 입찰 방식 | 기간 | 사유 |

| 2주차 | **비용 상한 (Cost Cap)** | D8~D14 | 학습 완료 후 CPI 상한 설정하여 비용 효율 확보. |

#### 메타 세부 설정

| 항목 | 설정 | 사유 |
|------|------|------|
| 캠페인 목표 | 앱 설치 (App Installs) | 직접 전환 최적화 |
| 최적화 대상 | 앱 설치 이벤트 | CPI 최적화 |

| 예산 유형 | 일 예산 (Daily Budget) | 일별 소진 관리 용이 |

#### CPI 목표 및 비용 상한 설정

```
1주차 (학습):
- 입찰: 최저 비용 (제한 없음)
- 예상 CPI: 2,500~4,000원 (학습 중 비용 높을 수 있음)
- 학습 완료 기준: 광고 세트당 주 50설치 이상
  → 일 예산 약 50,000원일 때 CPI 3,000원이면 약 17설치/일 → 주 119설치
  → 50설치 기준 달성 가능

2주차 (최적화):
- 입찰: 비용 상한 (Cost Cap)
- CPI 상한: 3,000원
- 학습 완료 후 효율적 노출
- CPI가 상한 아래로 안정화되면 예산 +20% 증액 검토

주의:
- 학습 중(1주차) 입찰/예산/타겟 변경 최소화
- 예산 변경 시 1회 20~30% 이내
- 광고 세트 수: 2~3개 (타겟 세분화, 과도한 분할 금지)
```

### 3-3. 300만원으로 1,000설치 가능성 분석

#### 결론: **가능하나 타이트. 적극적 최적화 필수.**

```
목표: 1,000설치 / 300만원 = 목표 CPI 3,000원

가능성 분석:

[낙관 시나리오] 평균 CPI 2,200원 → 약 1,364설치 (목표 136% 달성)
- 조건: 메타/구글 CPI가 2,000원 이하로 안착, 2주차 최적화 성공
- 확률: 20~25%

[기본 시나리오] 평균 CPI 3,000원 → 약 1,000설치 (목표 100% 달성)
- 조건: 채널별 CPI가 예상 범위 내, 2주차 재배분 적절 수행
- 확률: 40~45%

[비관 시나리오] 평균 CPI 4,000원 → 약 750설치 (목표 75% 달성)
- 조건: 학습 기간 CPI 높음, 최적화 효과 미미
- 확률: 25~30%

종합 성공 확률 (1,000설치 달성): 약 55~60%
```

#### 1,000설치 달성을 위한 핵심 조건

1. **메타 CPI 2,500원 이하 안착**: 메타가 전체 설치의 35%를 담당. CPI 관리가 핵심.
2. **구글 UAC 학습 성공**: tCPI 3,000원 내 학습 완료 시 안정적 설치 확보.
3. **1주차 데이터 기반 신속한 재배분**: D7 시점에 채널별 CPI 비교 후 24시간 내 재배분 실행.
4. **A/B 테스트 승자 빠른 확정**: 고성과 카피에 예산 집중으로 CPI 추가 절감.
5. **저효율 채널 과감한 축소**: CPI 5,000원 초과 채널은 예산 50% 이상 축소.

#### 목표 미달 시 대응 방안

| 시점 | 조건 | 대응 |
|------|------|------|
| D7 (1주차 종료) | 누적 설치 < 350건 | 최고 효율 채널에 예산 50% 집중, 저효율 채널 최소 유지 |
| D10 (2주차 중반) | 누적 설치 < 600건 | 오너에게 보고: 추가 예산 50~100만원 요청 또는 기간 1주 연장 제안 |
